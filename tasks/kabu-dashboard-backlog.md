- [✅ A2] ヘルプの指標説明 `1D / 1W`・`3M / 1Y` → `前日 / 1週間`・`3ヶ月 / 1年`（INDICATOR_ITEMS）
- [ A3] PEG / ROE / PBR / FEPS など専門略語：初心者向けに「？」アイコンや1行の平易な補足を本文側にも（現状ツールチップ依存。スマホはホバー不可で読めない）
