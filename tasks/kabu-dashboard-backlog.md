# kabu-dashboard 改善バックログ

最終更新: 2026-06-04 / 起点: 本人「UI/UX全面改善」オーダー（分かりにくい表現の排除・スマホ省スペース・色/視認性・脱AI感）

正本ルール: ここが kabu の作業台帳。完了は `✅`、着手中は `🔨`、本人確認待ちは `⏸`。
方針: ターゲットは投資初心者・若年・スマホ中心。「私（たいし）は分かるがユーザーには伝わらない表現」を潰す。無駄な文字・無駄な幅を削る。AI感（機械的な定型文・専門略語の生のせ）を消す。

- [✅ A2] ヘルプの指標説明 `1D / 1W`・`3M / 1Y` → `前日 / 1週間`・`3ヶ月 / 1年`（INDICATOR_ITEMS）
- [✅ A3-1] 詳細パネルの全指標（PER/PBR/ROE/PEG/EPS/BPS等）に**タップ式「?」用語解説**を実装。用語辞書 GLOSSARY を正本化（page.tsx Grid2 ＋ infoDot/infoPop CSS）。スマホでホバー不可問題を解消
- [✅ A3-2] レポート銘柄カルテの「株価の値動き」「PER位置」に汎用 InfoDot の「?」を展開。GLOSSARYに両用語追加。ダッシュのカード/SP行は「一覧は一目・詳細で学ぶ」方針で詳細パネル側に集約（A3-1で対応済）
- [✅ A4] PER位置バー（big表示）に**「安値○○」「高値○○」を明示**＝両端の数字が何か一目で分かる（PerBandBar）。中央の「割安/中立/割高」と合わせ凡例なしで意味が伝わる
- [ A4] 「PER位置」「PER今期」等の指標名：初見でも意味が分かる短い添え書き or 凡例を常設
- [✅ A5-1] 市場名 `Prime/Standard/Growth` → `プライム/スタンダード/グロース`（バッジ・絞り込みボタン・SPプルダウン・ヘルプを一括。marketShort/page.tsx）
- [✅ A5-2] 外部リンクの紛らわしい英語表記を統一：`Buffett`→`バフェットコード`（人物と誤解）、`TV`→`TradingView`（テレビと誤解）。カード/詳細パネルの全リンク配列で統一

## カテゴリG：本人指摘（2026-06-04 第2弾）
- [✅ G1] **詳細ポップアップがタブ切替で閉じない** → タブ変更で必ず閉じる（useEffect[tab]→setDetailCode(null)）
- [✅ G2] **右端に縦の黒フェード（アバター下〜最下部・ライト/ダーク両方）** → 真因は **常時DOMにある .helpPanel / .judgmentPanel（右端固定・全高・ダーク）が閉状態 translateX(100%) でも box-shadow:-12px 0 40px の影だけ画面内に漏れていた**こと。影を *Open（開いた時）だけに移して解消。※html/bodyのlightTheme追従（背景透け対策）も併せて実施済み（別効用）
- [✅ G3] PCカードビューの指標(1ヶ月%/PER今期/PEG/来期売上%/ROE/営業利益率)に「?」用語解説。ポップはカード幅内に表示＝横はみ出しなし（cardMetricPop）
- [✅ G4] 並べ替え中の指標表示を改善：「PER直近1年での水準順(perPos)」では生PER倍ではなく**○倍＋割安/中立/割高**を表示し、並べ替えの意味と一致
- [✅ G5] **決算が近い順ソート**：J-Quants自動取得の nextAnnouncementDate（手動入力 earningsDates が優先）で実装＝手動入力不要。SPの並べ替えに「決算が近い順」を追加、ソート中は行右に決算日(月/日)を表示。未取得は末尾

- [✅ G6] **テーマ(ライト/ダーク)が保存されず毎回ダークに戻る** → 保存useEffectがマウント時に既定値で上書きしていた。themeLoaded refで読込後のみ保存。ライトで終了→次回もライト維持
- [✅ G7] **英語名の銘柄が間延びして見える**（Ｓｙｎｓｐｅｃｔｉｖｅ等の全角英数字）→ buildStockRowで表示名を半角化（halfWidthAscii・カタカナ/漢字は不変）
