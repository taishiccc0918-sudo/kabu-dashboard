正本ルール: ここが kabu の作業台帳。完了は `✅`、着手中は `🔨`、本人確認待ちは `⏸`。
方針: ターゲットは投資初心者・若年・スマホ中心。「私（たいし）は分かるがユーザーには伝わらない表現」を潰す。無駄な文字・無駄な幅を削る。AI感（機械的な定型文・専門略語の生のせ）を消す。

## カテゴリA：分かりにくい表現（ユーザー視点の言い換え）
- [✅ A1] レポート銘柄カルテの騰落チップ `1W/1M/3M/1Y` → `1週間/1ヶ月/3ヶ月/1年`＋「株価の値動き」ラベル追加（page.tsx 4691 / repChipsLabel CSS）
- [✅ A2] ヘルプの指標説明 `1D / 1W`・`3M / 1Y` → `前日 / 1週間`・`3ヶ月 / 1年`（INDICATOR_ITEMS）
- [✅ A3-1] 詳細パネルの全指標（PER/PBR/ROE/PEG/EPS/BPS等）に**タップ式「?」用語解説**を実装。用語辞書 GLOSSARY を正本化（page.tsx Grid2 ＋ infoDot/infoPop CSS）。スマホでホバー不可問題を解消
- [✅ A3-2] レポート銘柄カルテの「株価の値動き」「PER位置」に汎用 InfoDot の「?」を展開。GLOSSARYに両用語追加。ダッシュのカード/SP行は「一覧は一目・詳細で学ぶ」方針で詳細パネル側に集約（A3-1で対応済）
- [✅ A4] PER位置バー（big表示）に**「安値○○」「高値○○」を明示**＝両端の数字が何か一目で分かる（PerBandBar）。中央の「割安/中立/割高」と合わせ凡例なしで意味が伝わる
- [ A4] 「PER位置」「PER今期」等の指標名：初見でも意味が分かる短い添え書き or 凡例を常設
- [✅ A5-1] 市場名 `Prime/Standard/Growth` → `プライム/スタンダード/グロース`（バッジ・絞り込みボタン・SPプルダウン・ヘルプを一括。marketShort/page.tsx）
- [✅ A5-2] 外部リンクの紛らわしい英語表記を統一：`Buffett`→`バフェットコード`（人物と誤解）、`TV`→`TradingView`（テレビと誤解）。カード/詳細パネルの全リンク配列で統一
